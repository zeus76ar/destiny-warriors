Antes de comenzar, debes prepararte. Por eso, tu primer combate sera de entrenamiento, para conocer la mecanica del juego. Tu oponente sera Lowell si elegiste a Kiara o viceversa.

Para combatir debes elegir alguna de las opciones: "atacar", "defender" y "tecnica especial" en cada turno del combate. Con cada ataque, defensa o especial, gastas tu "Energia" (tecnica especial consume m치s). Para recuperar energia, debes elegir la opcion "Defender".

Lo otro a tener en cuenta es la "Salud". Cada ataque que recibas reducira tu salud de acuerdo al da침o del ataque. Lo mismo para tu oponente. Para recuperar salud, debes elegir la opcion "Defender".

La "Tecnica Especial" se puede usar una vez que has recibido la cantidad especifica de ataques. Para c/personaje es distinta.

Como veras, la clave es atacar y defenderte de acuerdo a como vaya el combate. Para ganar un combate, se deben cumplir ciertas condiciones. Para ver cuales son, elige la opcion "Condiciones".
